# future-work-clarifie-v3.0

未来工作厘定 v3.0 —— AI 时代工作重构方法论与工具集

## 项目结构

本仓库包含两个并行版本：

```
├── generic/       # 通用版（适用于任意行业/岗位）
│   └── references/   # 方法论文档
└── hr/            # HR 版（面向人力资源模块）
    ├── agent-pack.yaml  # Agent 配置入口
    ├── agent/         # Agent 灵魂文件（SOUL/IDENTITY/AGENTS）
    ├── skills/        # 技能包（future-workshop / coe-modeling / hrbp-inventory / idp-design）
    ├── references/    # 参考材料
    └── release/       # 发版记录
```

## 两个版本的关系

| 维度 | 通用版 `generic/` | HR 版 `hr/` |
|------|-------------------|-------------------|
| 适用对象 | 任意企业/岗位的高管、中层 | HR 管理层、HRBP、OD |
| 核心文档 | 《AI落地通用方法论》 | 《未来工作厘定-HR版-V3.0-项目说明》 |
| Agent Pack | 待补充 | `hr/agent-pack.yaml` |
| 技能包 | 待补充 | `hr/skills/` 下 4 个技能 |

## 快速开始

- **通用版**：从 `generic/references/AI落地通用方法论.md` 开始阅读
- **HR 版**：从 `hr/agent/` 和 `hr/skills/future-workshop/` 开始

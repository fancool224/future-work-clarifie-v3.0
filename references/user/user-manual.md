# 未来工作厘定助手 - 用户说明手册（草案）

## 这个助手是做什么的？
它不是一个普通的 HR 问答工具。

它的作用是：
**带着 HR 管理层和模块负责人，一起把某个 HR 相关工作拆开来看，判断它在 AI 时代会怎么变、哪些工作会被接走、哪些工作会升值、哪些问题要重新设计。**

## 谁适合用？
- HRD
- HRBP负责人
- OD负责人
- 招聘/ER/TD/绩效/薪酬负责人
- 一起参与共创的业务管理者

## 使用前要准备什么？
### 最少准备
1. 这次想讨论哪个工作
2. 当前主要职责是什么
3. 最耗时间的工作是什么
4. 最难、最依赖经验的工作是什么
5. 当前最想解决的问题是什么
6. 有哪些角色会参与讨论

### 如果想效果更好，建议再准备
- 职责说明文件
- 工作清单表格
- 典型流程或案例

## 使用时会做什么？
助手会带着你分五步走：
1. 先看清这个工作本质上是什么类型
2. 把它拆成不同工作颗粒度
3. 判断这些颗粒度分别属于规则、风险、诊断、设计还是共识问题
4. 推演 AI 会怎么介入
5. 提炼这个工作未来会从什么转向什么

## 最后你会得到什么？
1. 这个工作的颗粒度地图
2. 哪些工作会被 Agent 接走
3. 哪些工作会被显著增强
4. 哪些工作未来会更值钱
5. 哪些问题不是单角色能解决，而要跨角色共创
6. 这个工作未来应该往哪里走

## 一句话理解
它不是帮你回答一个 HR 问题，
而是帮你搭出一张“这个工作未来怎么变”的地图。

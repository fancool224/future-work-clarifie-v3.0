# HRBP 人才盘点

## 盘点全流程

### 阶段1：初始化盘点
### 阶段2：素质评分处理（异常检测 → IQR去极值 → 组内中位数 → 全局基准 → 调平）
### 阶段3：落位计算 + 报告输出（5个Sheet：25格矩阵/员工明细/梯队名单/数据质量报告/统计摘要）
### 阶段4：进度提醒（HRBP侧，不直接联系业务团队）

## 调平算法

严格顺序：① 标记异常评分者 → ② 基于非异常数据计算IQR → ③ 标记极值
使用 median + 3×MAD 替代 mean + 2σ，对离群值鲁棒
有效样本为0时回退到全量原始数据，标记 fallback=True

## 关键约束

- 每次盘点用命名隔离
- 异常数据不删除，只排除出调平计算
- KPI取值规则按部门配置
- 落位规则每次盘点开始前确认

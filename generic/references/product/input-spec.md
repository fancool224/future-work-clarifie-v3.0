# 通用版输入材料规范

> 材料是加速器，不是门槛。先聊，再补材料。

---

## 最小启动包（3 项，随时可开始）

只要能说清楚以下三点，就能启动共创：

1. **想讨论什么工作** — 一个岗位 / 一项具体工作 / 一个痛点
2. **当前最大的困扰是什么** — 为什么现在想讨论这个
3. **你是从什么视角来用的** — 管理者（看团队/部门）还是个体（看自己）

---

## 结构化输入（建议在拆解阶段补充）

当 Agent 进入 S2 颗粒度拆解阶段时，建议用户提供以下信息：

### 输入 1：工作职责说明（Word/PPT/文本即可）
建议包含：
- 当前核心职责（3-8 条）
- 汇报关系
- 关键协作对象
- 主要考核指标

### 输入 2：工作清单（Excel/CSV/文本即可）
建议包含：
- 日常工作项（名称 + 频次 + 耗时）
- 周期性工作项（名称 + 周期 + 耗时）
- 非常规但关键的工作项

### 输入 3：流程或典型案例（SOP/文档/文本即可）
建议包含：
- 1-2 个典型工作流程
- 关键节点和决策点
- 当前瓶颈或卡点

---

## 工具表入口

如果用户说不清工作清单或颗粒度边界：
→ 引导使用 `references/tools/work-inventory-template.xlsx`

这张表的作用：
- 作为工作盘点入口
- 作为颗粒度拆解支架
- 作为后续 S3-S6 结构化判断的输入底座

**使用时机**：S2 颗粒度拆解阶段，用户表达较散或颗粒度边界说不清时。

---

## 材料原则

1. **不要求用户整理成特定格式** — 原始文档/文字/口述都可以
2. **材料不完整也可以开始** — Agent 在对话中逐步引导补全
3. **材料是加速器** — 有材料能更快进入深层判断，没材料也能从对话开始
4. **不因缺材料而卡住** — 如果用户暂时提供不了，Agent 基于已有信息继续推进
